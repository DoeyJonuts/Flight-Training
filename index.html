Read this before using the timer!!!

Rather than picking up your controller and immediately queueing up for Unranked games, maybe you should get a warm-up in? Flight Training is a tool for Falco players to quickly and conveniently run through relevant tech skill before starting to grind versus real players.

Before using the timer, there's a few things you should note. The tool is designed to quickly run through tech skill that you are already somewhat familiar with. This tool is not meant for learning new tech for the first time. If there's specific tech on the list that you haven't performed before, you should probably take separate time outside of the timer to work on the mechanics of that tech.

If you find yourself messing up specific tech while the timer is ticking, you can choose to highlight that tech by clicking the save icon so that it is easy to revisit during the last 3 minutes of the timer (click again to unhighlight). This is by far the most important part of regular tech skill training. You want to avoid mindlessly spamming tech skill and deliberately look for the tech you've been dropping lately so that you can focus on improving what you're bad at.

There's one more thing before you should give the timer a try. If the tech you're practicing involves hitboxes (Aerials, Jump Shine Waveland, etc), it's highly recommended that you practice those on a character to properly train your timing.
